       <h3>Accessibility Commitment</h3> 
       <p>The Social Security Administration is committed to making our programs, benefits, services and facilities, and information and communications technology accessible to everyone, in accordance with Section 504 of the Rehabilitation Act of 1973, and Section 508 of the Rehabilitation Act (29 U.S.C. 794d), and relevant implementing regulations. For more information, please refer to the following overviews:</p> 
       <p> <a href="504_overview.html" class="underline">Section 504 Overview</a><br> <a href="508_overview.html" class="underline">Section 508 Overview</a> </p> 
       <h3>BrowseAloud</h3> 
       <p>BrowseAloud is browser plug-in for people who find it difficult to read online. By reading text aloud and highlighting the words as they are spoken, BrowseAloud can help people with learning disabilities, English as a second language, and mild visual impairments.</p> 
       <p><a href="browseAloud.html" class="underline" title="Learn more about BrowseAloud screen reader software">Learn more</a></p> 
       <h3>Web Accessibility Help</h3> 
       <div class="asbAccordion" role="navigation"> 
        <ul class="accordionUL"> 
         <li> <a class="topLevel" href="#"><span class="accordionHeader">I can't see very well</span></a> 
          <div class="accordionSub" tabindex="0"> 
           <p>If you have trouble seeing web pages, the following information explains how to use your computer or web browser to make it easier to see web pages on ssa.gov, and have web pages read out loud to you.</p> 
           <p> <a href="browseAloud.html">Use your computer to read web pages out loud</a><br> <a href="textsize.html">Increase Text Size</a><br> <a href="magnifyscreen.html">Magnify your screen</a><br> <a href="changecolors.html">Change background and text colors</a><br> <a href="mousepointer.html">Make your mouse pointer more visible</a><br> <a href="error_messages.html">Respond to warning messages</a><br> </p> 
          </div>
          <!-- close .accordionSub --> </li> 
         <li> <a class="topLevel" href="#"><span class="accordionHeader">I am blind</span></a> 
          <div class="accordionSub" tabindex="0"> 
           <p>If you are blind, the following information explains how to use your computer, web browser and screen reader assistive technology to navigate web pages and online services on ssa.gov.</p> 
           <p> <a href="keyboard_nav.html">Use the Keyboard to Navigate Screens</a><br> <a href="forms_nav.html">Navigate Forms</a><br> <a href="title_attribute.html">Get Link Details Using Title Attributes</a><br> <a href="error_messages.html">Respond to Warning Messages</a><br> <a href="confirmation_numbers.html">Save &amp; Print Forms and Confirmation Numbers</a> </p> 
          </div>
          <!-- close .accordionSub --> </li> 
         <li> <a class="topLevel" href="#"><span class="accordionHeader">I find a keyboard or mouse hard to use</span></a> 
          <div class="accordionSub" tabindex="0"> 
           <p>If you find a keyboard or mouse difficult to use, the following information explains how to use speech recognition software to navigate web pages and online services on ssa.gov.</p> 
           <p>Speech recognition software allows the user to move focus to an application object by voicing the object label name or the object type.</p> 
           <p>Object types include:</p> 
           <ul class="bullet"> 
            <li>Text Box</li> 
            <li>List Box</li> 
            <li>Link</li> 
            <li>Image</li> 
            <li>Screen</li> 
            <li>File</li> 
           </ul> 
           <br> 
           <p>Each of these object types are given a label name that is visible on the screen and the voice user can say the name to move focus to that object.</p> 
           <p>A screen, an application or file can be accessed by voicing the text in the Title Bar.</p> 
           <p>If there is an object or many objects on a screen without labels then the user can voice an object type which will number all objects of that type on that screen. For example voicing "link" would number all links that are on the visible screen and then a specific link can then be accessed by choosing the number desired. Voicing "checkbox", "radio button, "text box, etc. will operate in the same manner.</p> 
           <p>If the screen design does not allow moving focus by voicing a label or an object type then the speech recognition software may provide a utility to move the mouse pointer in discrete steps or continuously. </p> 
           <p>Social Security has designed web pages and applications to utilize several different methods of using speech in order to provide flexibility in using speech recognition capabilities. Users should try combinations of these methods to determine what best meets an individual's needs.</p> 
           <p> <a href="trainingDragon.html">How to train Dragon NaturallySpeaking</a><br> <a href="error_messages.html">Respond to warning messages</a> </p> 
          </div>
          <!-- close .accordionSub --> </li> 
         <li> <a class="topLevel" href="#"><span class="accordionHeader">I am deaf or hard of hearing</span></a> 
          <div class="accordionSub" tabindex="0"> 
           <p>If you are deaf or hard of hearing, there are several accessibility features available to you.</p> 
           <p class="underline">Accessibility features</p> 
           <ul class="bullet"> 
            <li>volume control</li> 
            <li>transcripts</li> 
            <li>captioning</li> 
            <li>all information and alerts conveyed via audio are also conveyed visually</li> 
           </ul> 
           <br> 
           <p class="underline">Volume control</p> 
           <ul class="bullet"> 
            <li>click the Start button</li> 
            <li>click "Control Panel"</li> 
            <li>click "Hardware and Sound"</li> 
            <li>click "Adjust System Volume"</li> 
            <li>use the volume control to increase or decrease sound to the desired level</li> 
           </ul> 
           <br> 
           <p class="underline">Transcripts</p> 
           <p>A text transcript is a text equivalent of audio information that includes spoken words and non-spoken sounds such as sound effects.</p> 
           <p class="underline">Captioning</p> 
           <p>A caption is transcript for the audio track of a video presentation that is synchronized with the video and audio tracks. Captions are generally rendered visually by being superimposed over the video, which benefits people who are deaf and hard-of-hearing, and anyone who cannot hear the audio (e.g., when in a crowded room).</p> 
           <p>The following information explains how to use captioning for video on demand and for both live and recorded webinars.</p> 
           <p>For video on demand, click the link to start the video playing. Next, click on the Closed Captioning (CC) button to activate the captioning feature.</p> 
           <p>For some videos and webinars, open captions are used. In these cases, the captioning feature cannot be turned off.</p> 
           <p>The Windows Media Player is required for online video content. If this media player is already installed on your computer, the player will come up automatically. If you do not have the player, it can be downloaded here: <a href="http://windows.microsoft.com/en-US/windows/downloads/windows-media-player" class="underline disclaimer" target="_blank" rel="noopener noreferrer">Microsoft Media Player download</a>.</p> 
          </div>
          <!-- close .accordionSub --> </li> 
        </ul>

        <h3>Requesting Assistance</h3> 
        <h4>General Assistance</h4> 
        <p>By calling <span class="bold">1-800-772-1213</span>, you can use our automated telephone services to get recorded information and conduct some business 24 hours a day. If you cannot handle your business through our automated services, you can speak to a Social Security representative between 7 a.m. and 7 p.m. Monday through Friday. Generally, you'll have a shorter wait time if you call during the week after Tuesday. If you are deaf or hard of hearing, call our toll-free TTY number, <span class="bold">1-800-325-0778</span>, between 7 a.m. and 7 p.m. Monday through Friday. See <a href="http://www.ssa.gov/pgm/services.htm" class="underline">Automated Services</a> for more details.</p> 
        <h4>Web Accessibility Problems</h4> 
        <p>Please report website accessibility problems to the Social Security webmaster, <a href="mailto:webmaster@ssa.gov" class="underline">webmaster@ssa.gov</a>. In your message, include the website address or URL and the specific problems you have encountered.</p>
